Why HTML beats markdown for complex knowledge

Each of these artefacts was built because the information had structure that prose can't show — tabs that organize without burying, tables that invite comparison, interactive decisions that activate rather than describe. The page you're reading is itself one of those artefacts.

These aren't pretty wrappers. Every one was built to answer a real question faster than scrolling a report.
